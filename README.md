# 陪玩小程序

## 1.介绍

这是一个仿比心的陪玩微信小程序，主要是用原生微信小程序做客户端和node的express框架做后台，Python写脚本爬游戏资讯，mysql为数据库

## 2.安装

1. 找到文件夹下的server，使用npm install 安装相关依赖，然后使用node bin/www启动服务器。
2. 打开微信开发者工具，打开项目中的play文件夹，即可使用。
